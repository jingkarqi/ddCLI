## ddCLI 开发文档
### 项目概述
ddCLI是一个智能命令行工具，用户可以通过自然语言（如 `dd 删除当前文件夹` ）来执行复杂的终端命令。工具会将自然语言转换为实际的终端命令并执行。

### 技术架构
- 核心语言 : PowerShell
- AI集成 : 支持多种LLM API（ChatGPT、DeepSeek等）
- 配置管理 : JSON格式存储API密钥和用户偏好
- 安全机制 : 命令预览和用户确认机制

### 功能模块
- 自然语言解析 : 利用LLM API将用户输入的自然语言转换为命令
- 命令执行 : 执行转换后的命令并返回结果
- 配置管理 : 允许用户配置API密钥、模型偏好等
- 安全机制 : 对生成的命令进行预览并要求用户确认执行

### 具体实现命令

#### 核心命令
1. `dd <自然语言命令>` : 解析用户输入的自然语言并执行相应的命令
   - 示例: `dd 删除当前目录下的所有txt文件`
   - 示例: `dd 创建一个名为test的新文件夹`
   - 支持管道操作和复杂指令

#### 配置相关命令
2. `dd config [选项]` : 配置管理
   - `dd config show` : 显示当前所有配置
   - `dd config reset` : 重置为默认配置

3. `dd security [选项]` : 安全设置管理
   - `dd security whitelist` : 管理命令白名单

#### 系统命令
4. `dd help [命令]` : 帮助系统
   - `dd help` : 显示总体帮助信息
   - `dd help <命令>` : 显示特定命令的详细帮助
   - `dd help examples` : 显示常用示例

5. `dd version` : 版本信息
   - 显示当前版本号
   - 显示更新日志
   - 检查更新状态

6. `dd setting` : 交互式设置
   - 使用上下箭头选择设置项

7. `dd history` : 命令历史
   - 显示历史命令记录
   - 统计使用频率

### 安全机制详细设计

#### 危险命令检测
- 文件系统操作：rm, del, format, mkfs
- 系统管理：shutdown, reboot, format
- 网络相关：netsh, iptables
- 权限提升：sudo, runas

#### 用户确认流程
1. 生成命令后显示完整预览
2. 高亮显示潜在危险操作
3. 要求用户输入 'yes' 或 'y' 确认/或者使用上下箭头选择确认
4. 提供取消选项

### 错误处理机制

使用标准报错确保用户能够及时发现并解决问题

#### AI API错误
- 网络连接失败
- API密钥无效
- 请求频率限制
- 模型不可用

#### 命令执行错误
- 命令不存在
- 权限不足
- 参数错误
- 系统资源不足
